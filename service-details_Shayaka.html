Request and Request Tracking Overview

Features

  • Displays all requests submitted by the user in a categorized format. Users can filter requests based on their status:

    • All: Lists all requests.

    In Progress: Shows requests currently being processed.

    Completed: Displays successfully fulfilled requests.

    Rejected: Includes requests that have been denied.

    Search Functionality

    Users can easily search for specific requests using keywords such as

  • Request name.
  • Request category.
  • Status.
    • Detailed Request Information

    Users can view complete details of each request, including

  • Requester's name, age, and gender.
  • Unique tracking ID (Sahayak Number).
  • Contact details, such as email and address.
  • Status updates with color-coded badges:

    • Green for "Completed."

    Yellow for "In Progress."

    Red for "Rejected."

    Acknowledgment Action
  • A “Thank You” button for users to acknowledge or close completed requests.
    • Request History
  • A section that allows users to view past requests and their outcomes.
  • Organized in a card layout with summary details like
  • Name, age, category, and status

Reward Overview

Features

    1. How get reward
  • Users submit a help request (medical, educational, farmer aid, etc.)
  • Each approved request (post-Karma’s verification) earns 10 score points.
  • Target Example: 100 points = 10 approved requests.
  • Fraudulent/incomplete requests are rejected (0 points).
  • Reward History: Displays a list of past rewards earned by the user, including details like gift cards for platforms such as Flipkart and Swiggy etc.
  • Reward Tracking: Users can see their total scores and total rewards and track their progress toward the next reward milestone.
  • Reward Tracking: Users can see their total scores and total rewards and track their progress toward the next reward milestone.

She is Safe Overview

Features

The She is Safe Module is a comprehensive safety feature specifically designed for the protection of girls and women. This module ensures swift assistance during emergencies with the following functionalities:

    Emergency SOS Button:
  • Press the SOS button to instantly share your live location with the Karmas Karmas Will Verify your Emergency and Shares nearest police station and NGOs that work for women’s safety.
  • Ensures immediate support during critical situations.
    • On-Screen Widget:
  • TQuickly access the SOS button through a widget displayed on your mobile screen for instant help.
    • Add Trusted Contacts:
  • Easily include friends, family, or Shayaka members as trusted contacts for emergency notifications.

    • This module empowers women with tools to enhance their safety and ensure immediate help is available whenever needed.
